Title: Realtime

================================================================================

Realtime
Send and receive messages to connected clients.
Supabase provides a globally distributed
Realtime
service with the following features:
Broadcast
: Send low-latency messages between clients. Perfect for real-time messaging, database changes, cursor tracking, game events, and custom notifications.
Presence
: Track and synchronize user state across clients. Ideal for showing who's online, or active participants.
Postgres Changes
: Listen to database changes in real-time.
What can you build?
#
Chat applications
- Real-time messaging with typing indicators and online presence
Collaborative tools
- Document editing, whiteboards, and shared workspaces
Live dashboards
- Real-time data visualization and monitoring
Multiplayer games
- Synchronized game state and player interactions
Social features
- Live notifications, reactions, and user activity feeds
